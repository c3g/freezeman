  </b>
</p>



## Installation

```bash
# Clone the repository
git clone https://github.com/c3g/freezeman
cd freezeman
git submodule update --init --recursive
```

Next, follow the instructions in [/backend/](./backend/) and
[/frontend/](./frontend/).
